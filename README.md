# Music-Manager
Hello, my name is Sidharth V. Here are the list of all the important things to keep in mind while
operating on this CLI Music player:
1. Albums are the equivalent to Playlists, and queue is where you play all the songs you want.
2. If you want to play any song, you must first add it into a queue
3. To add a song into the queue, it must first be present in any one of the albums.
4. The songs must be added from the library only, otherwise you cannot add into the album.
5. For simplicity sake, I have named the Songs as alphabets in uppercase, and Artists as alphabets in lowercase.
   Do refer to the masterfile to see the list of all the songs you can add!
6. You cannot play an album as such, you have to add all of the songs in the album into the queue manually.
7. The commands have been assigned a number, you have to enter the number corrseponding to the command for it to be in effect
8. Please be careful while entering inputs, if a number is asked, make sure to enter a number only.
